# Hoje vai ter? 👀

Um mini-site romântico/brincadeira feito com HTML, CSS e JavaScript.

## 💡 Ideia
Mostra a pergunta: **"Hoje vai ter?"** com dois botões:
- "Sim" → leva para um vídeo especial 😏
- "Não" → o botão fica fugindo quando tenta clicar
